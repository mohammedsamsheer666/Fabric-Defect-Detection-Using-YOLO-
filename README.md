# Fabric Defect Detection Using YOLO

This project is an AI-based fabric defect detection system developed using YOLO to identify defects in fabric images.

## Features
- Detects fabric defects automatically
- Uses YOLO model for object detection
- Image annotation using LabelImg
- Streamlit interface for prediction

## Technologies Used
- Python
- YOLO
- Streamlit
- LabelImg
- Kaggle

## How to Run
1. Install requirements
2. Run:
streamlit run app.py
